A Lost Ring

A while back as a birthday present for my sister, I made a real life treasure hunt. The theme of this hunt was for a ring of power from Lord of the Rings.

To test out the hunt I built a web version of the hunt using Ink and Inky. It’s quite similar to, and was inspired by, my other treasure hunt, Champlain’s Treasure.

You can find the digital version of A Lost Ring here.

Champlains Treasure

A while back I made a treasure hunt for a friend to solve. This involved hiding random clues around their house and leading them from one clue to another. While I was making the hunt, I found myself wanting to test it out without remaking physical clues all the time.

I decided to use Ink and Inky to help me create a digital version of the hunt - a proof of concept. With Ink I was able to test for coherency and see if it would be fun. The difficulty was another story because it’s easy to solve the hunt when dealing with multiple choice.
